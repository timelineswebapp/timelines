# SEO Architecture

Authority Level: Architecture
Governed System: Metadata, sitemap, robots, and JSON-LD.
Describes: Both

## Scope
Documents implemented SEO surfaces.

## Non-Scope
Does not define keyword strategy.

## Verified Implementation
Public pages generate metadata. Sitemap includes static pages, categories, tags, timelines, and milestones. Robots disallows API and admin guess paths. Timeline and milestone JSON-LD are generated by `src/lib/timeline-jsonld.ts`.

## Future Architecture
SEO should account for publication states, empty tags, canonical merges, deprecations, and source credibility.

## Dependencies
`app/sitemap.ts`, `app/robots.ts`, `src/lib/timeline-jsonld.ts`, `src/lib/social-metadata.ts`.

## Open Questions
- Should orphan tags be excluded from sitemap?

## Future Evolution Guidance
Only index records with meaningful public content and stable canonical identity.
