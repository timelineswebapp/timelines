TiMELiNES Institutional Architecture

Version: 1.0

Status: LOCKED

Authority Level: Tier 1 Constitutional

⸻

Purpose

This document defines the permanent institutional architecture of TiMELiNES.

It establishes institutional ownership, authority boundaries, publication flow, and the movement of historical knowledge from research to public publication.

This document is constitutional authority.

Implementation shall conform to this architecture.

⸻

Foundational Principle

TiMELiNES is an institutional historical publishing system.

Historical knowledge progresses through a sequence of independent institutions.

Each institution owns a specific responsibility.

Authority flows forward.

Authority is never recreated downstream.

Institutional Responsibilities

1. Factory

Factory owns historical knowledge generation.

Factory is responsible for:

* topic discovery;
* topic prioritization;
* research;
* source discovery;
* source validation;
* evidence extraction;
* historical object extraction;
* milestone extraction;
* participation extraction;
* relationship extraction;
* context generation;
* publication candidate preparation.

Factory owns Production Memory.

Factory never publishes.

Factory never bypasses Governance.

⸻

2. Historical Publication Package

The Historical Publication Package is the constitutional transfer artifact between Factory and Governance.

It contains:

* candidate Historical Objects;
* candidate Milestones;
* candidate Participations;
* candidate Relationships;
* candidate Context;
* candidate Sources;
* validated Evidence references;
* artifact lineage;
* governance lineage;
* publication lineage.

It is immutable once submitted.

⸻

3. Governance

Governance owns editorial authority.

Governance evaluates canonical historical knowledge.

Governance may:

* approve;
* reject;
* request revision.

Governance never creates historical knowledge.

Governance never modifies historical facts.

Governance never publishes directly.

⸻

4. Historical Library

Historical Library owns canonical preservation.

Historical Library admits only Governance-approved authority.

Historical Library owns:

* canonical persistence;
* publication versions;
* historical preservation;
* publication audit;
* authority lifecycle.

Historical Library never invents history.

Historical Library never performs editorial review.

⸻

5. Published Memory

Published Memory owns immutable published historical authority.

Published Memory represents the officially published state of historical knowledge.

Published Memory preserves:

* canonical authority;
* governance decisions;
* publication lineage;
* evidence lineage;
* provenance.

Published Memory never edits history.

⸻

6. Projection

Projection owns deterministic transformation.

Projection derives:

* Timeline Views;
* Timeline Membership;
* Search Documents;
* SEO Documents;
* Platform Read Models;
* API read models.

Projection creates presentation only.

Projection never creates authority.

Projection never modifies historical knowledge.

⸻

7. Platform Runtime

Platform Runtime owns user experience.

Platform Runtime consumes Projection.

Platform Runtime owns:

* APIs;
* public website;
* search interface;
* timeline rendering;
* milestone rendering;
* administration UI.

Platform Runtime never creates historical authority.

Platform Runtime never modifies canonical knowledge.

Canonical Historical Authority

Canonical historical authority consists of:

* Historical Objects;
* Milestones;
* Participations;
* Relationships;
* Context Records;
* Sources;
* Citations;
* Governance Decisions;
* Publication Versions.

Milestones are the canonical chronology-bearing authority.

In the current implementation, Milestones are persisted as Events.

Timeline Views are derived.

Timeline Membership is derived.

Law 1

Every historical fact originates in Factory.

⸻

Law 2

⸻

Law 5

Published Memory contains only admitted canonical authority.

⸻

Projection never derives authority.

⸻

Platform Runtime consumes Projection.

Platform Runtime never owns historical authority.

⸻

Authority is never regenerated downstream.

Every institution consumes authority created by upstream institutions.

⸻

Law 9

Evidence, provenance, governance decisions, and publication lineage shall be preserved throughout the institutional chain.

⸻

Law 10

No institution may bypass another institution defined by this architecture.
