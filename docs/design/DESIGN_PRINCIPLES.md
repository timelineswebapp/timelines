## Non-Scope

## Verified Implementation
The UI favors glass surfaces, compact cards, timeline streams, serif headings, sans body text, and restrained admin interfaces.

## Ownership Boundaries
`docs/design/DESIGN_SYSTEM_CONSTITUTION.md` is the primary design authority. This document owns practical principles for visual and interaction decisions beneath that constitution.

Foundation, component, pattern, accessibility, and brand documents own their specific domains only when consistent with the design constitution.
