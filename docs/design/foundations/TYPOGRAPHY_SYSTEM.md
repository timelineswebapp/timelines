# Typography System

Authority Level: Design Foundation
Governed System: Typography.
Describes: Both

## Scope
Documents type foundations.

## Non-Scope
Does not define copywriting guidelines.

## Verified Implementation
Root layout loads `Cormorant Garamond` for serif display and `Plus Jakarta Sans` for sans text.

## Future Architecture
